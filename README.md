# Manhunt
逃げる側と追いかける側に別れて競うゲーム

## コマンド説明

コマンドは基本OP必須です

### 基本

* `/manhunt[mh] group <人数>`  
ランナーを`人数`分グルーピングする

* `/mh start <ミッション>`  
`ミッション`をランナーが達成したら終了のマンハントを開始する

* `/mh end <ミッションを達成したことにするかどうか>`  
マンハントを強制終了する

### データベース

* `manhuntstatus[mhstats] <プレイヤー> {日付}`  
`プレイヤー`のステータスを見る。`日付`を指定するとその日のデータも見れる

* `/manhuntmatch[mhmatch] <日付>`  
`日付`の試合のデータを見る
